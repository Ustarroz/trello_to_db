##README##

quickstart est un script en js qui permet de recuperer les information contenu dans un tableur google

##ETAPES A RESPECTER##
* Creer un compte google
* Initialiser l'api google pour l'utilisation de google sheet
* Mettre en place une authentification Oauth2
* Recuperer les information de connexion (client_secret.json)
* Creer une premiere connection et generer un fichier d'autorisation (credential)
